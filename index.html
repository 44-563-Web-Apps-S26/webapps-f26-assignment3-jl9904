How to make a pillow.

  1. Gather your materials:
    1. Fabric (For ease its helpful if both sides of the fabric are the same)
    2. Thread
    3. Scissors
    4. Pillow stuffing
  2. Cut 2 pieces of fabric to the desired size.
  3. Place the pieces pretty sides together.
  4. Stitch the edges together, leaving one side with an open hole.
  5. Turn the pillow right side out.
  6. Stuff the pillow with filling.
  7. Hand sew the open hole closed.

INNER JOIN : SQL

An INNER JOIN in SQL is used to combine rows from two or more tables based on a related column between them. It returns only the rows where there is a match in both tables. For example, if we have a "Customers" table and an "Orders" table, an INNER JOIN would return only the customers who have placed orders.

Here is an example SQL query using INNER JOIN:

SELECT column_name(s)
            FROM table1
            INNER JOIN table2
            ON table1.column_name = table2.column_name;
